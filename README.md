# C Programming Practice

A collection of basic C programming programs created to strengthen problem-solving skills and understand core programming concepts.

## Overview

This repository contains beginner-friendly C programs covering fundamental topics such as:

- Input and Output
- Variables and Data Types
- Operators
- Conditional Statements
- Loops
- Functions
- Arrays
- Strings
- Pointers
- Structures
- File Handling
- Basic Algorithms

## Requirements

- GCC Compiler or any standard C compiler
- Code Editor (VS Code, CodeBlocks, Dev-C++, etc.)

## How to Compile and Run

Compile the program:

```bash
gcc program.c -o program
```

Run the executable:

```bash
./program
```

### Windows

```bash
gcc program.c -o program.exe
program.exe
```
